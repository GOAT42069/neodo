# Main objective
Survive as long as you can and get as much points as you can.

---

## Controls
* Keyboad:
  * WASD: move
  * Space: dash
* Mobile:
  * Left stick: move
  * Right button: dash

Dashing has a small cooldown and it leaves behind you ghost, if a object collides with your ghost you get a "GHOST KILL".

---

## Objects
<li>&#x1F535; FOLLOWS you around</li>
<li>&#x1F7E3; FAST projectile</li>
<li>&#x1F7E0; EXPLODES and kills you if in that radius</li>
<li>&#x1F7E2; SLOWS you if you are in its range</li>
